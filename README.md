# chimmyShop 
## The Odin Project : Shopping Cart
Shopping Cart project to practice routing with react router. Fake products with [Fake Store API](https://fakestoreapi.com/).

## Features
- Display all products
- Search products by category
- Add items to cart
- Update cart
- Responsive
